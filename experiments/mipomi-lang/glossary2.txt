Mipomi Glossary
1. Core CV Roots
These are the fundamental building blocks of the language.
self / feeling: MA (nurture), MI (self), MO (feel / love)
living beings: BA (life / child), BI (peer / kin), BO (creature)
possession / transfer: PA (keep / hold), PI (take / receive), PO (give)
nature / element: FA (fire / heat), FI (earth / soil), FO (sky / air)
thing / place: KA (object / stuff), KI (tool / device), KO (place / container)
beauty / design: LA (light / good), LI (fine / detail), LO (shape / form)
time / goal: TA (past), TI (now), TO (future)
voice / story: JA (speak), JI (name / ID), JO (you / listener)
negation / rest: NA (none / absence), NI (rest / sleep), NO (end / negate)
body / substance: DA (body / flesh), DI (eat / drink), DO (stone / rock)
mind / pattern: SA (idea), SI (sign / tag), SO (pattern / structure)
action / control: HA (start), HI (go / act), HO (stop / halt)
water / qualities: WA (water / flow), WI (cold / ice), WO (round / solid)
2. Function Words (Grammar)
I / me: mi (root)
be (is/are): e (particle)
go / move: a (particle)
do / act: o (particle)
